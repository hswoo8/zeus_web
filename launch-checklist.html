2. 순서대로 진행

  1. 게임명 후보 3~5개를 만든다. 완료: 미니제우스.
  2. KIPRIS, Google Play, App Store, Steam, 도메인 검색으로 충돌을 확인한다.
  3. 최종 게임명과 패키지명 방향을 확정한다. 완료: MiniZeus, com.bakingstudio.minizeus.
  4. 도메인 1개 이상을 확보한다. 완료: https://www.minizeusgame.com/.
  5. 상표 출원 여부를 결정한다. 오래 쓸 이름이면 출원을 권장한다.
  6. 가족 명의 Google Play 개발자 계정을 준비한다.
  7. 가족 명의 사업자등록 필요 여부를 확인한다.
  8. 관할 구청에 게임제작업/게임배급업 등록 필요 여부와 자택 등록 가능 여부를 문의한다.
  9. Git 기록, 원본 이미지, AI 프롬프트, 수정 과정, 에셋 라이선스를 백업한다.
  10. 개인정보처리방침, 공식 홈페이지, 문의 이메일을 준비한다.
  11. Play Console에서 내부 테스트와 비공개 테스트를 진행한다.
  12. 문제가 없으면 공개 출시한다.

3. 앱 내부 작업

  • Android applicationId를 com.bakingstudio.minizeus로 변경
  • Debug 빌드의 .debug suffix 유지
  • 앱 표시명을 미니제우스로 유지
  • 앱 내부 공식 사이트 URL을 https://www.minizeusgame.com/로 변경
  • 앱 내부 개인정보처리방침/문의 화면의 이름과 링크 정리
  • AAB 빌드 준비
  • :android:assembleDebug 빌드 검증
  • 가능하면 Android 기기 설치/실행 검증
  • 첫 출시 빌드에 광고, 인앱결제, 채팅, 유저 생성 콘텐츠, 서버 저장 기능이 없는지 확인

4. 앱 외부 작업

  • 웹사이트 문구를 미니제우스와 MiniZeus 기준으로 통일
  • 개인정보처리방침 URL을 https://www.minizeusgame.com/privacy.html로 정리

5. 신청·등록 항목

항목 필요도 명의 준비물 메모
Google Play 개발자 계정 필수 가족 Google 계정, 실명, 주소, 전화번호, 결제수단, 인증 정보, 공개 연락처 스토어의 공식 게시자 기준이 된다.
상표 출원 강력추천 가족 게임명, 영문/한글 표기, 로고 파일, 특허고객번호, 지정상품/서비스 최소 9류 게임 소프트웨어, 41류 온라인 게임 서비스를 검토한다.
사업자등록 확인 필요 가족 신분증, 사업자등록신청서, 임차한 경우 임대차계약서 광고, 인앱상품, 굿즈, 후원 수익화 전에는 세무 확인을 권장한다.
게임제작업 등록 공개 출시 전 확인 권장 가족 등록신청서, 영업소 서류, 제작시설 및 장비 명세서 개인 인디 모바일 게임도 관할 구청 확인 후 진행하는 편이 안전하다.
게임배급업 등록 공개 출시 전 확인 권장 가족 등록신청서, 영업소 서류, 사업계획서 Google Play 공개 유통을 가족이 담당한다면 가족 명의로 문의한다.
저작권 등록 선택 가족 또는 실제 권리자 대표 캐릭터, 로고, 일러스트, 제작 과정 자료 등록하지 않아도 저작권은 발생하지만, 분쟁 시 증거력 확보에 도움이 된다.

6. 관할 구청 문의 문구

아래 문구를 그대로 복사해서 문의한다.

개인 인디 모바일 게임을 가족 명의로 Google Play에 무료 출시하려고 합니다. 광고와 인앱결제는 현재 없지만 추후 추가 가능성이 있습니다. 이 경우 가족 명의 사업자등록 후 게임제작업과 게임배급업 등록을 해야 하나요? 자택 주소로 등록 가능한지와 필요서류도 알고 싶습니다.

7. 권리 방어용 백업

  • Git 커밋 기록
  • 최종 빌드 파일과 출시 전 테스트 영상
  • AI 프롬프트, 생성 날짜, 생성 원본
  • 캐릭터 수정 전/후 이미지
  • PSD, Procreate, Clip Studio 등 레이어 원본
  • 게임에 사용한 최종 PNG, 사운드, 폰트, 라이선스
  • Play Console 등록 화면과 출시일 캡처

8. 9~12세 타겟과 추후 광고

9~12세를 타겟에 포함하면 나중에 광고를 붙일 수는 있지만, 어린이 대상 정책이 같이 따라온다. 그래서 첫 출시에는 광고, 인앱결제, 채팅, 회원가입, 유저 생성 콘텐츠를 넣지 않는 현재 방향이 안전하다.

추후 광고를 붙일 때 확인할 것

  • Google Play Families 정책 적용 여부
  • 어린이 대상에 허용되는 광고 SDK인지 여부
  • 맞춤형 광고 제한과 연령별 광고 설정
  • 개인정보처리방침 수정
  • Data Safety 수정
  • Play Console의 광고 포함 여부 표시
  • EU/영국 이용자 대상 동의 관리 필요 여부

운영 난이도만 보면 13세 이상이 더 편하지만, 9~12세도 포함하려면 광고를 나중에 붙일 때 보상형 광고 위주로 신중하게 검토하는 것이 좋다.

9. 현재 게임 내용 기준

  • 피 없음
  • 잔인한 연출 없음
  • 시체, 절단 표현 없음
  • 현실 무기 느낌 약하게 유지
  • 번개, 마법, 판타지 타격 중심
  • 채팅 없음
  • 유저 생성 콘텐츠 없음
  • 첫 출시 버전 광고 없음
  • 첫 출시 버전 결제 없음
